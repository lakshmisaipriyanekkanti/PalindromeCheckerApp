# PalindromeCheckerApp
// running procedure
Program starts
A hardcoded string is initialized
The string is reversed
The original string is compared with the reversed string
Result is displayed in the console
Program exits

//objective
To design and implement a Java program that:
Checks whether a given string is a palindrome
Uses fundamental Java concepts
Demonstrates basic string manipulation and conditional logic

//use case
UC2: Print a Hardcoded Palindrome Result
Goal:
Display whether a hardcoded string is a palindrome.
Actor:
User (runs the program)
Flow:
Program starts
Hardcoded string is checked
Result is printed
Program exits
